Code Demo

Get Up and Hike

Get Up and Hike is a full stack app built in the PERN (Postgres, Express, React, NodeJs) stack. It is a community style blog designed for outdoors lovers. Users can read recent blog posts and create and delete posts written by them. It has register and login functionality and checks to see if the user who is logged in matches the post author before deleting.

Gift Exchange Randomizer

This app is built in HTML, CSS, and JavaScript. This is a random pairing gift exchange generator where a user can add a list of names for a gift exchange and then randomize the list pairing up partners from the list inputed.
